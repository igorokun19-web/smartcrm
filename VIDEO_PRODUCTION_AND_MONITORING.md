## 🎥 VIDEO PRODUCTION WORKFLOW

### **Hardware Needed:**
- ✅ Laptop/Desktop (already have)
- ✅ Camera/Phone (for intro shots) - optional
- ✅ Microphone (built-in is fine, external is better)
- ✅ Screen recording software

### **Software (Free):**
1. **Screen Recording:** OBS Studio (free, open source)
   - Download: https://obsproject.com/
   - Record your screen + audio
   - Output as MP4
   
2. **Video Editing:** CapCut (free)
   - Download: https://www.capcut.com/
   - Or DaVinci Resolve (free version)
   - https://www.davinciresolve.com/

3. **Audio:** Audacity (free)
   - Clean up background noise
   - https://www.audacityteam.org/

### **Step-by-Step Recording Process:**

**STEP 1: Setup (10 minutes)**
```
1. Open OBS Studio
2. Add scene "Desktop Recording"
3. Add sources:
   - Display Capture (your screen)
   - Audio Input Capture (microphone)
4. Start recording
5. Open MyServices CRM in browser
```

**STEP 2: Record Each Section (5-10 minutes per video)**
```
For Video 1 ("How to Add a Lead"):
1. Start OBS recording
2. Read intro to camera (or use voiceover)
3. Record screen: Go to Leads
4. Record: Click "Add Lead"
5. Record: Fill in form
6. Record: Click Save
7. Read outro
8. Stop recording
9. File saved as video.mp4
```

**STEP 3: Basic Editing (15 minutes per video)**
```
In CapCut or DaVinci:
1. Import MP4 file
2. Trim beginning/end
3. Add title card at start: "How to Add a Lead"
4. Add music (royalty-free from YouTube library)
5. Speed up slow parts (optional)
6. Add text captions for key points
7. Export as MP4 (1080p)
```

**STEP 4: Quality Check**
```
Watch video:
- Audio clear? ✓
- Visuals visible? ✓
- No long pauses? ✓
- Total time correct? ✓
Ready to upload!
```

## 🎬 RECORDING TIPS

1. **Microphone Quality:** Use headphones to hear yourself
2. **Screen Resolution:** Zoom in UI so it's visible (150-200%)
3. **Pacing:** Speak slowly and clearly (people watch at 1.25x speed)
4. **Pauses:** Pause 2 seconds between sections (makes editing easier)
5. **Mistakes:** Keep recording, fix in editing
6. **Lighting:** Natural light from window is fine
7. **Background:** Plain wall or nice office is better than messy desk
