### STEP-BY-STEP VIDEO 1: "The Truth About Lost Leads"

**Time Budget: 60 minutes total**
- Setup: 10 min
- Script Review: 5 min
- Recording (2-3 takes): 15 min
- Editing: 20 min
- Uploading: 10 min

#### STEP 1: SETUP (10 minutes)

**Clean your space:**
```
✅ Desk tidy
✅ No distracting items behind you
✅ Lighting from front/side (not behind)
✅ Phone/camera on tripod (DIY: books stack)
✅ Microphone close (but not in frame)
✅ White balance correct (not too yellow/blue)
```

**Test recording:**
```
1. Open OBS Studio
2. Click "Start Recording"
3. Talk for 10 seconds
4. Click "Stop Recording"
5. Play back - do you sound clear?
   - If muffled: Move mic closer
   - If loud background: Go to quieter room
   - If echo: Add blanket/pillow to reduce reflections
```

#### STEP 2: REVIEW SCRIPT (5 minutes)

**Use this from CONTENT_PRODUCTION_MASTER_PLAN.md:**

Video 1: "The Truth About Lost Leads" (7 minutes)

```
[0:00-0:15] HOOK
"עובדה: 43% מהלידים שאתה מפסיד כל שבוע?
הם כבר בדוא"ל או בוואטסאפ שלך.

היום אני מראה למה..."

[0:15-0:45] PROBLEM VISUALIZATION
...
[Continue reading through entire script]
```

**Tip:** Print script on paper. Have in front of you while recording.

#### STEP 3: RECORD (15 minutes)

**Camera Position:**
```
✅ Sit at desk, camera at eye level
✅ 2-3 feet away
✅ Good lighting on your face
✅ Background visible but not distracting
```

**Recording Tips:**
```
🎯 Speak clearly and slowly
🎯 Pause between sentences (easier to edit)
🎯 If you mess up, pause 2 seconds then re-start
🎯 Confidence is better than perfection
🎯 Record 2-3 takes (pick best later)
```

**How to Record in OBS:**
```
1. Click "Start Recording" (button at bottom)
2. Start talking through script
3. If mess up: Pause, wait 2 seconds, restart sentence
4. Finish complete video
5. Click "Stop Recording"
6. File saved to Documents/Videos by default
```

#### STEP 4: ADD SCREEN RECORDINGS (If needed for demo)

**For showing features, use screen recording:**

```
1. Open OBS Studio
2. Click "+" under Sources
3. Select "Display Capture" (or "Game Capture")
4. Select your monitor
5. Click "Start Recording"
6. Narrate while showing website
7. Click "Stop Recording"
```

#### STEP 5: EDIT VIDEO (20 minutes - Optional but recommended)

**Using DaVinci Resolve (FREE):**

```
1. Download: https://www.blackmagicdesign.com/davinci-resolve
2. Open program
3. File → New Project
4. Drag video file into timeline
5. Add title at start:
   - Right-click timeline
   - Add → Title
   - Type your title
6. Add graphics/text overlays (optional but professional)
7. Export → YouTube format (1080p)
```

**OR Use CapCut (Mobile - Easier):**
```
1. Open CapCut app
2. New project
3. Add your video
4. Tap "Text" to add title/graphics
5. Tap "Music" to add royalty-free audio (optional)
6. Tap "Export" (select 1080p)
7. Done
```

**Quick Edits:**
```
✅ Add title at beginning (3 sec)
✅ Add graphics showing statistics
✅ Add text overlay of key points
✅ Cut any dead air (long pauses)
✅ Add royalty-free background music (optional)
```

## 💡 TIPS FOR PROFESSIONAL QUALITY

### Audio Quality:
```
✅ Speak clearly
✅ Reduce background noise (close windows)
✅ Use microphone (even headphone mic helps)
✅ Normalize audio in editing (prevents loud/soft)
✅ Add royalty-free background music (subtle)
```

### Video Quality:
```
✅ Film at 1080p minimum (60fps for YouTube shorts)
✅ Good lighting (front light, not backlit)
✅ Steady camera (tripod or phone stand)
✅ Clean background
✅ Face to camera when introducing
✅ Show screen recordings when explaining
```

### Editing Quality:
```
✅ Remove long pauses
✅ Add title at start (3 seconds)
✅ Add graphics during key points
✅ Include subtitles (helps retention)
✅ Add CTA at end (10 seconds)
✅ Consistent branding (colors, fonts, logo)
```
