## 🎥 PRODUCTION WORKFLOW

### Simple 3-Step Process:

**Step 1: Record (15 minutes)**
```
1. Open OBS Studio (free)
2. Record your screen
3. Follow script from document
4. Narrate as you go
```

**Step 2: Edit (20 minutes)**
```
1. Open CapCut (free, mobile app)
2. Add title (3 sec)
3. Add text overlays
4. Add captions (Hebrew + English)
5. Export 1080p
```

**Step 3: Publish (10 minutes)**
```
1. Upload to YouTube
2. Add description
3. Add tags
4. Create thumbnail
5. Publish
```

**TOTAL TIME: 45 minutes per video**
